Guide: Changing Permissions

This guide tells how to check and make changes to permissions. Follow the guide from 1-3 and check the “Uh oh” section for additional help.


Hide your task by clicking the down arrow in the button right corner and scroll down


  1. Check whats enabled in "Current Permissions"! Check marks indicate that permissions are enabled!
    1. “Select A User” and select the user
    2. Click on “Permissions” next to the file you wish to check. You can click “OK” when done with this step!
    3. Look at the Effective Permissions under the Effective Permissions Section. A permission with a check is enabled for the user on the file you selected.
  2. Change permissions
    1. Scroll down to look at the section in Blue (has a folder icon)
    2. Click on the “Permissions” button
    3. Select User/Group
    4. “Deny”: checking a permission in the deny column DENIES permissions for user, overriding “Allow” on that permission
    5. “Allow” checking a permission in the deny column GIVES permissions to user
    6. To be safe, if you are removing permissions for a user, use “Deny”
  3. Click “OK” to implement changes!
    1. Repeat Step 1: Check which permissions are enabled!
